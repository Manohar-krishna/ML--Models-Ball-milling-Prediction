Project Scope: Inputs & Outputs

The project's objective is to create a machine learning model that predicts the final properties of γ-TiAl powder based on the ball milling process settings. This section outlines the controllable inputs the model uses and the powder characteristics it aims to predict.

Model Inputs (Process Parameters)

  • Ball-to-Powder Ratio (BPR)
  • Milling Speed (RPM)
  • Milling Duration (Hours)
  • Process Control Agent (PCA)

Model Outputs (Powder Properties)

  • Particle Size (e.g., d₅₀)
  • Particle Morphology (e.g., Sphericity)

The Data Acquisition Bottleneck

Feasibility & The Data Gap

The project's greatest challenge is acquiring a large, high-quality dataset. Machine learning models require substantial data to train effectively, but generating this data for materials science is a major hurdle.

Key Insight

Due to the data scarcity, this project is best framed as a methodological proof-of-concept. Its primary value is in demonstrating the workflow and challenges of applying ML in a data-limited environment, rather than delivering a production-ready tool.

Significance: A Shift in Manufacturing

This data-driven approach marks a critical shift from slow, costly trial-and-error methods towards an intelligent "Materials 4.0" paradigm, accelerating development and deepening scientific understanding.

Novelty: A Unique Intersection

The project's innovation isn't a new algorithm, but the challenging synthesis of three distinct domains. It tackles a high-value, difficult-to-process material system where data is exceptionally scarce, distinguishing it from prior work.

Future Trajectories

This foundational work opens several exciting avenues for more advanced research, addressing the project's limitations and expanding its impact significantly.

1. Explainable AI (XAI)

Move beyond prediction to explanation. Use techniques like SHAP to understand *why* the model makes its predictions, turning it into a tool for scientific discovery and hypothesis generation about the underlying physics.

2. Hybrid Simulation-Experiment

Overcome data scarcity by pre-training a model on a large dataset from physics-based simulations (DEM), then fine-tuning it with a small set of real experimental data. This "transfer learning" approach maximizes accuracy with limited resources.

3. Through-Process Modeling

Create a multi-stage digital thread linking the entire manufacturing chain: from initial milling parameters to powder characteristics, then to consolidated part properties, and finally to the final component's mechanical performance.
